# SEE App – Smart Emotional Enhancement

**Graduation Project – Team Project**

SEE App is a cross-platform mobile application developed using **Flutter** and **Firebase**, designed to support child mental health. The app features an **AI-powered therapist assistant**, **parent-therapist dashboards**, and **emotional analytics**, focusing on usability, data security (HIPAA/GDPR), and scalability.

## Key Features
- AI-driven emotional insights
- Real-time interaction and monitoring
- Multi-role support (children, parents, therapists)
- Scalable architecture for large user base

## Technologies & Tools
- Flutter (Dart)
- Firebase (Authentication, Firestore)
- Python (for AI modules)
- GitHub for version control
